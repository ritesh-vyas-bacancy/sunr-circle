📋 Overview

This guide walks you through setting up the SUNR Circle project on a new Windows laptop with Android Studio. After completing all steps, you will be able to run the Flutter mobile app on an Android emulator or physical device, and the Next.js admin panel locally.

🛠️ Step 1 — Install Prerequisites

Install the following tools on the new laptop. The order matters.

#ToolVersionDownload URLNote

4Android StudioLatest✅ Already installedRun flutter doctor to link

Flutter SDK Installation

1. Download the Flutter zip from: docs.flutter.dev/get-started/install/windows

2. Extract to C:\flutter (do NOT put in Program Files)

3. Add Flutter to PATH:

# Run in PowerShell as Administrator
[System.Environment]::SetEnvironmentVariable(
  "PATH",
  $env:PATH + ";C:\flutter\bin",
  "Machine"
)

4. Restart PowerShell/Terminal after adding to PATH

🔧 Common Issues & Solutions

Flutter — "flutter" is not recognized

Flutter is not in PATH. Run in PowerShell as Administrator:

[System.Environment]::SetEnvironmentVariable(
  "PATH", $env:PATH + ";C:\flutter\bin", "Machine"
)

Then restart the terminal completely.

Admin Panel — "This page couldn't load"

This is usually an environment variable issue. Verify your .env.local file has all the required values from Step 3. Restart the dev server after editing.

Flutter — No connected devices

1. Open Android Studio → Device Manager → Start an AVD emulator, OR
2. Connect an Android phone via USB → Enable USB Debugging in Developer Options, OR
3. Run flutter devices to list available devices

Admin Panel — Next.js build error

cd apps/admin
npm install       # Reinstall dependencies
npm run build     # Test production build
