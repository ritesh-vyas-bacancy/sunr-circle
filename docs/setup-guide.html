Flutter SDK Setup on Windows

1. Download zip from docs.flutter.dev and extract to C:\flutter (no spaces in path, NOT in Program Files)

2. Add to Windows PATH (run PowerShell as Administrator):

[System.Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\flutter\bin", "Machine")
# RESTART PowerShell/Terminal after this command

3. Verify installation:

flutter --version
flutter doctor         # Should show all green checkmarks
flutter doctor --android-licenses    # Accept licenses once

E. Common Issues and Solutions

"flutter" is not recognized

# PowerShell as Administrator, then restart terminal:
[System.Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";C:\flutter\bin", "Machine")

Admin panel page shows error

Check .env.local has all values. Most common cause: missing or incorrect SUPABASE_SERVICE_ROLE_KEY. Restart dev server after editing env file.

No Android device found

flutter devices     # Lists all connected devices
# Open Android Studio > Device Manager > create and start an emulator
